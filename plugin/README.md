
# API-IDEE ChatAgent Plugin

An AI assistant plugin for the API-IDEE map viewer, providing Q&A capabilities about API-IDEE's functionality.

## Features

- Integrated chat interface within the map viewer.
- Communicates with a Django-based AI backend for RAG (Retrieval-Augmented Generation).
- Can answer questions about API-IDEE features, usage, and plugin development.
- Pure JavaScript, no build step required.

## Usage

1.  **Include the plugin JavaScript file** in your HTML after the API-IDEE library:

    ```html
    <script src="path/to/api-idee-js-library.js"></script>
    <script src="path/to/plugin/chatagent.js"></script>
    ```

2.  **Instantiate and add the plugin to your map**:

    ```javascript
    // Ensure IDEE is loaded before this script runs
    if (window.IDEE) {
      const map = IDEE.map({ container: 'map' /* ... other map options ... */ });

      const chatAgent = new IDEE.plugin.ChatAgent({
        position: 'TR',              // Panel position: 'TL', 'TR', 'BL', 'BR'
        collapsed: true,             // Start collapsed: true or false
        backendUrl: 'http://localhost:8000/api', // URL of your Django backend
        tooltip: 'Asistente API-IDEE', // Tooltip for the collapsed button
        placeholder: 'Pregunta sobre API-IDEE...', // Input field placeholder
        welcomeMessage: '¡Hola! Soy el asistente de API-IDEE. Pregúntame lo que quieras.' // Custom welcome
      });

      map.addPlugin(chatAgent);
    }
    ```

## Configuration Parameters

| Parameter      | Type    | Default                      | Description                                   |
| :------------- | :------ | :--------------------------- | :-------------------------------------------- |
| `position`     | `string`| `TR`                         | Panel position on the map (`TL`, `TR`, `BL`, `BR`) |
| `collapsed`    | `boolean`| `true`                       | Whether the panel starts collapsed            |
| `backendUrl`   | `string`| `http://localhost:8000/api`  | URL of the Django backend API                 |
| `tooltip`      | `string`| `Asistente API-IDEE`         | Tooltip text for the plugin button            |
| `placeholder`  | `string`| `Pregunta sobre API-IDEE...` | Placeholder text for the chat input field     |
| `welcomeMessage`| `string`| (default text)               | Initial welcome message in the chat           |

## Development

This plugin is developed as a single, self-contained JavaScript file (`chatagent.js`) with inlined CSS and HTML, requiring no build tools (like Webpack or NPM) for deployment. It adheres to API-IDEE's plugin architecture for seamless integration.
